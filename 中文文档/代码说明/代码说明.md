# 交易监控系统 - 代码说明

## 项目概述

本系统用于监控 Solana 链上指定 Token（通过 Mint 地址）的实时交易数据。
核心功能：输入 Mint 地址 → 回填历史交易 → 实时 WebSocket 监听 → 逐条处理 → 前端展示。

## 核心模块说明

### 1. 历史回填服务 (trade_backfill.py)

**职责**: 获取指定 Mint 地址的全部历史交易数据并入库。

**流程**:
1. 记录当前 Slot 作为分界点（S0）
2. 使用 `getSignaturesForAddress` 分页获取所有交易签名（每页 1000 条）
3. 使用 Helius 增强 API `/v0/transactions` 批量解析交易（每批 100 条，并发 3 路）
4. 批量入库（INSERT OR IGNORE 去重）
5. 回填完成后，获取增量数据补漏
6. 切换到实时流

**效率优化**:
- 批量获取：100 条交易/次 API 调用（而非逐条查询）
- 并发控制：通过 `asyncio.Semaphore` 限制并发数
- 批量入库：减少数据库写入次数
- 可配置参数：batch_size、concurrent_requests、request_interval

### 2. 实时流服务 (trade_stream.py)

**职责**: 通过 Helius WebSocket 订阅实时交易数据。

**流程**:
1. 连接到 `wss://api.helius.xyz/v0/stream`
2. 发送订阅请求（按 Mint 地址过滤）
3. 持续接收新交易消息
4. 解析交易信息 → 入库 → 触发处理 → 推送前端
5. 断线自动重连（3 秒后）

### 3. 交易处理引擎 (trade_processor.py)

**职责**: 逐条处理交易，计算净流入、价格、钱包标记等。

**当前状态**: 框架已搭建，具体计算逻辑待补充。

**预留接口**:
- `process_trade()`: 处理单条交易
- `calculate_metrics()`: 计算四个核心指标（本轮下注、本轮成本、已落袋、交易笔数）

### 4. 设置服务 (settings_service.py)

**职责**: 管理系统运行时参数。

**默认设置**:
| 参数 | 默认值 | 说明 |
|------|--------|------|
| batch_size | 100 | 批量解析交易数量 |
| concurrent_requests | 3 | 并发请求数 |
| request_interval | 0.5 | 批次间隔（秒） |
| ws_max_connections | 10 | WebSocket 最大连接数 |
| trade_page_size | 50 | 前端显示最大交易行数 |

### 5. WebSocket 管理器 (websocket/manager.py)

**职责**: 管理所有 WebSocket 连接，支持按 Mint 地址分组广播。

**功能**:
- 连接/断开管理
- 按 Mint 广播消息
- 自动清理失效连接
- 全局单例 `ws_manager`

### 6. 前端页面 (templates/trade_monitor.html)

**布局**:
- 顶部：Mint 输入框 + 开始/关闭按钮 + 状态显示
- 中部：四个指标卡片（本轮下注、本轮成本、已落袋、交易笔数）
- 底部：实时交易表格（最多 50 行，最新在上，超出滚动）

**实时更新**:
- 通过 WebSocket 接收新交易数据
- 新行高亮动画（fadeIn）
- 定期刷新指标数据

## 数据流

```
用户输入 Mint → 点击开始
    │
    ├─ 记录 current_slot = S0
    ├─ 状态 → BACKFILLING
    ├─ 分页回填 (getSignaturesForAddress → /v0/transactions → 批量入库)
    ├─ 状态 → CATCHING_UP (补漏)
    ├─ 状态 → STREAMING (WS 实时监听)
    └─ 用户点击关闭 → 状态 → STOPPED
```

## 状态机

```
IDLE → BACKFILLING → CATCHING_UP → STREAMING → STOPPED
```

## API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/admin/trade-monitor` | GET | 交易监控页面（登录后默认首页） |
| `/admin/settings-page` | GET | 系统设置页面 |
| `/admin/api/trades/{mint}` | GET | 获取交易列表 |
| `/admin/api/trades/{mint}/metrics` | GET | 获取四个核心指标 |
| `/admin/api/trades/{mint}/start` | POST | 启动监听 |
| `/admin/api/trades/{mint}/stop` | POST | 停止监听 |
| `/admin/api/settings` | GET | 获取所有设置 |
| `/admin/api/settings/{key}` | PUT | 更新设置 |
| `/admin/ws/trades/{mint}` | WebSocket | 实时推送 |

## 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `HELIUS_API_KEY` | Helius API 密钥 | 无 |
| `HELIUS_NETWORK` | Solana 网络 | mainnet-beta |
| `DATABASE_URL` | 数据库连接字符串 | sqlite:///./gmgn.db |

- 后台管理页面：`http://localhost:8000/admin/trade-monitor`（需要登录）
