# 指数计算公式详解

## 一、数据结构

### 1.1 用户状态（Redis user:{address}）

| 字段 | 初始值 | 说明 |
|------|--------|------|
| `status` | unknown | 用户状态：unknown / dealer / retail |
| `holdingQty` | 0 | 当前持有代币数量 |
| `holdingCost` | 0 | 当前持仓成本（SOL） |
| `avgPrice` | 0 | 持仓均价 = holdingCost / holdingQty |
| `totalBuyAmount` | 0 | 累计买入总金额（SOL） |
| `totalSellAmount` | 0 | 累计卖出总金额（SOL） |
| `totalSellPrincipal` | 0 | 累计卖出本金（SOL） |

### 1.2 全局指标（Redis metrics:{mint}）

| 字段 | 初始值 | 说明 |
|------|--------|------|
| `total_bet` | 0 | 全池总投入（所有用户持仓成本之和） |
| `realized_profit` | 0 | 全池已实现总收益 |
| `dealer_count` | 0 | 庄家数量 |

## 二、完整示例：3轮买卖的完整数据流

### 场景设定
- 代币单位：Token
- 用户初始状态：0 Token，0 SOL
- 持仓均价初始：0

---

### 第1步：买入 100 Token，花费 1 SOL

**交易数据：**
```
tx_type = "BUY"
sol_spent = 1.0
amount = 100
```

**用户状态计算：**
```
buy_amount = 1.0 SOL
buy_qty = 100 Token

holdingQty = 0 + 100 = 100
holdingCost = 0 + 1.0 = 1.0 SOL
totalBuyAmount = 0 + 1.0 = 1.0 SOL
avgPrice = 1.0 / 100 = 0.01 SOL/Token
```

**全局指标变化：**
```
delta_bet = 1.0 - 0 = 1.0  →  total_bet += 1.0
delta_profit = 0 - 0 = 0   →  realized_profit += 0
```

**计算后数据：**
```
用户状态：
  holdingQty = 100
  holdingCost = 1.0 SOL
  avgPrice = 0.01 SOL/Token
  totalBuyAmount = 1.0 SOL
  totalSellAmount = 0
  totalSellPrincipal = 0

全局指标：
  total_bet = 1.0
  realized_profit = 0
```

---

### 第2步：再买入 100 Token，花费 3 SOL

**交易数据：**
```
tx_type = "BUY"
sol_spent = 3.0
amount = 100
```

**用户状态计算：**
```
buy_amount = 3.0 SOL
buy_qty = 100 Token

holdingQty = 100 + 100 = 200
holdingCost = 1.0 + 3.0 = 4.0 SOL
totalBuyAmount = 1.0 + 3.0 = 4.0 SOL
avgPrice = 4.0 / 200 = 0.02 SOL/Token
```

**全局指标变化：**
```
delta_bet = 4.0 - 1.0 = 3.0  →  total_bet += 3.0
delta_profit = 0 - 0 = 0     →  realized_profit += 0
```

**计算后数据：**
```
用户状态：
  holdingQty = 200
  holdingCost = 4.0 SOL
  avgPrice = 0.02 SOL/Token
  totalBuyAmount = 4.0 SOL
  totalSellAmount = 0
  totalSellPrincipal = 0

全局指标：
  total_bet = 4.0
  realized_profit = 0
```

---

### 第3步：卖出 100 Token，获得 5 SOL

**交易数据：**
```
tx_type = "SELL"
sol_spent = -5.0  (获得)
amount = -100     (减少)
```

**关键概念：卖出本金**
- **卖出本金** = 卖出数量 × 当前持仓均价
- 本金是"相当于最初投入了多少 SOL 买的这批币"
- 公式：`sell_principal = sell_qty × avgPrice`

**用户状态计算：**
```
sell_qty = 100
sell_amount = 5.0 SOL
sell_principal = 100 × 0.02 = 2.0 SOL  ← 卖出本金

totalSellPrincipal = 0 + 2.0 = 2.0 SOL
totalSellAmount = 0 + 5.0 = 5.0 SOL
holdingQty = 200 - 100 = 100
holdingCost = 4.0 - 2.0 = 2.0 SOL
avgPrice = 2.0 / 100 = 0.02 SOL/Token (不变)
```

**落袋收益 = 卖出收入 - 卖出本金**
```
realized = 5.0 - 2.0 = 3.0 SOL  ← 这才是真正赚到的
```

**全局指标变化：**
```
old_holding_cost = 4.0
new_holding_cost = 2.0
old_realized = 0 - 0 = 0
new_realized = 5.0 - 2.0 = 3.0

delta_bet = 2.0 - 4.0 = -2.0  →  total_bet -= 2.0 (减少，因为卖出了)
delta_profit = 3.0 - 0 = 3.0  →  realized_profit += 3.0
```

**计算后数据：**
```
用户状态：
  holdingQty = 100
  holdingCost = 2.0 SOL
  avgPrice = 0.02 SOL/Token
  totalBuyAmount = 4.0 SOL
  totalSellAmount = 5.0 SOL
  totalSellPrincipal = 2.0 SOL
  realized（落袋）= 5.0 - 2.0 = 3.0 SOL

全局指标：
  total_bet = 4.0 - 2.0 = 2.0
  realized_profit = 0 + 3.0 = 3.0
```

## 三、核心公式汇总

### 3.1 买入公式

```
buy_amount = |sol_spent|
buy_qty = amount

holdingQty += buy_qty
holdingCost += buy_amount
totalBuyAmount += buy_amount
avgPrice = holdingCost / holdingQty
```

### 3.2 卖出公式

```
sell_qty = |amount|
sell_amount = |sol_spent|
sell_principal = sell_qty × avgPrice  ← 关键：用均价计算本金

totalSellPrincipal += sell_principal
totalSellAmount += sell_amount
holdingQty -= sell_qty
holdingCost -= sell_principal
avgPrice = holdingCost / holdingQty (如果 holdingQty > 0)

realized（落袋） = sell_amount - sell_principal
```

### 3.3 全局指标公式

```
delta_bet = new_holding_cost - old_holding_cost
delta_profit = new_realized - old_realized

total_bet += delta_bet
realized_profit += delta_profit
```

## 四、最终输出指标

### 4.1 API 返回的 4 个核心指标

在 `calculate_metrics` 函数（trade_processor.py:546-564）中：

| 指标名 | 公式 | 含义 |
|--------|------|------|
| `current_bet` | total_bet | 当前全池总投入 |
| `current_cost` | total_bet - realized_profit | 当前全池剩余持仓成本 |
| `realized_profit` | realized_profit | 全池已实现总收益 |
| `trade_count` | COUNT(Transaction) | 交易总笔数 |

### 4.2 各指标含义

| 指标 | 计算方式 | 含义 |
|------|----------|------|
| `current_bet` | total_bet | 市场总投入，代表锁仓深度 |
| `realized_profit` | realized_profit | 所有用户落袋收益之和 |
| `current_cost` | total_bet - realized_profit | 未变现的持仓成本（≈当前未卖出仓位的价值） |
| `trade_count` | COUNT | 市场活跃度 |
