## 3. 核心函数

### `check_c003()` - 检测单笔交易

```python
def check_c003(tx_data: dict, min_cu=105000, max_cu=115000):
    cu = meta.get("computeUnitsConsumed", 0)
    # 判断 CU 是否在 105k-115k 范围内
    return min_cu <= cu <= max_cu
```

### `fetch_transactions()` - 获取钱包交易

```python
# 获取最近 10 笔交易（不只是第一笔）
sortOrder: "desc"  # 从最新到最旧
limit: 10          # 获取 10 笔
```

### `test_wallet()` - 测试单个钱包

```python
# 遍历所有交易，寻找有 C003 的交易
for tx in txs[:10]:
    if check_c003(tx)["in_range"]:
        has_c003 = True
        c003_count += 1

# 如果有任何一笔交易 CU 在 105k-115k，判定为庄家
detected = "dealer" if has_c003 else "retail"
```

## 4. 测试流程

```
┌─────────────────────────────────────┐
│  遍历 16 个庄家钱包                 │
│  遍历 11 个散户钱包                 │
└─────────────────────────────────────┘
           ↓
┌─────────────────────────────────────┐
│  对每个钱包：                       │
│  1. 获取最近 10 笔交易             │
│  2. 检查每笔交易的 CU              │
│  3. 统计 CU 在 105k-115k 的数量   │
│  4. 如果 >= 1 笔，判定为庄家       │
└─────────────────────────────────────┘
           ↓
┌─────────────────────────────────────┐
│  汇总结果：                         │
│  - 庄家正确识别：11/16             │
│  - 散户正确识别：5/11              │
└─────────────────────────────────────┘
```

## 5. 检测逻辑

| 步骤 | 操作 |
|------|------|
| 1 | 调用 Helius API 获取钱包交易 |
| 2 | 遍历每笔交易提取 `computeUnitsConsumed` |
| 3 | 判断是否在 **105,000-115,000** 范围 |
| 4 | 统计命中数量 `c003_count` |
| 5 | `c003_count >= 1` → 判定为庄家 |
